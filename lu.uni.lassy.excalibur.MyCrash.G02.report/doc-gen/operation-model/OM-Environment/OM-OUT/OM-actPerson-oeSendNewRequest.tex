\label{OM-oeSendNewRequest}


The \msrcode{oeSendNewRequest} operation has the following properties:

	\begin{operationmodel}
	\addheading{Operation}
	\adddoublerow{oeSendNewRequest}{sent to add a new request to the systemâ€™s post state.}

	\addrowheading{Parameters}
	\addnumbereddoublerow{}{APIName: dtName}{used to initialise the name field} 
	\addnumbereddoublerow{}{APICategory: etCategory}{used to initialise the category field}

	\addsinglerow{ptBoolean}

	\addnumberedsinglerow{PreP}{the system is started.

	\addrowheading{Post-Condition (functional)}
	\addnumberedsinglerow{PostF}{the new requested PI is added in the system's post state.
	the person actor is informed about the satisfaction of its request.}

	\addrowheading{Post-Condition (protocol)}
	\addnumberedsinglerow{PostP}{none}
	\end{operationmodel}
